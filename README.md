# えいえんのアサリガール

  ブラウザで遊べるシンプルな縦スクロールシューティングゲームです。

  ## 遊び方

　https://asarigirl.github.io/STG/  
　容量は560MGくらい？
　初回ロード長い
　PC推奨
　スマホだと難易度高
　エンディングはランダムで４種類

  ## 操作方法

  ### PC

  -   移動: WASD または 矢印キー
  -   攻撃: スペース または 左クリック
  -   ULT (必殺技): 左Shift または 右クリック (ゲージが溜まると使用可能)
  -   一時停止: ESC

  ### スマートフォン

  -   移動: 画面をスワイプ
  -   攻撃: 自動
  -   ULT (必殺技): 画面のULTボタンをタップ
